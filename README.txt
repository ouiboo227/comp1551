
StringProcessing Console Application
====================================

Description:
------------
This C# console application processes a string consisting of capital letters (A-Z) and an integer shift value (N between -25 and 25).
It supports:
- Encoding the input string by shifting characters in the alphabet.
- Displaying ASCII codes before and after encoding.
- Sorting the string alphabetically.
- Encrypting and decrypting the string using AES encryption.

How to Use:
-----------
1. Run the executable (Coursework.exe).
2. Input a string (only uppercase letters) and a valid integer (-25 <= N <= 25).
3. Choose operations like encoding, viewing ASCII codes, sorting, encrypting, or decrypting the string.

Requirements:
-------------
- Windows 10 or later.
- .NET Framework 4.7.2 installed.

Build:
------
Built using Visual Studio targeting .NET Framework 4.7.2.

Notes:
------
- Ensure your inputs are valid to avoid validation errors.
- Encryption and decryption features are for demonstration only and are not intended for production use.
